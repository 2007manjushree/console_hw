Project on Console

Visual Code Studio is a compiler which converts high level languages to binary language so that the computer and read it. It is created by microsoft and is considered one of the best. You can go to their offical website to download it to your pc or mac. After downloading, you can run it and then use it!

Console screen is used to debug the code. You can acces it by right-clicking on any part of the web page and then clicking inspect. You can then choose console and see the console screen. You can print stuff in the console by using the "console.log()" command
